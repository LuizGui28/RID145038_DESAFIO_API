 # Clima e CEP - Consulta de clima + endereço
Projeto de aplicação web para consultar informações de clima em tempo real e dados de endereço atráves do CEP, consumindo APIs externas.

 # Funcionalidades: 
- Preenchimento de formulário com nome completo e e-mail
- Consulta de endereço a partir do CEP informado
- Consulta de dados climáticos utilizando latitude e longitude
- Exibição de informações de clima em tempo real
- Validação básica dos dados inseridos no formulário
- Integração com APIs externas para obtenção dos dados

 # Tecnologias aplicadas: 
- HTML 
- CSS 
- JavaScript 

 # APIs: 
- OpenWeather (clima) /
- ViaCEP (CEP)
